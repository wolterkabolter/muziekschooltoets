# Muziekschooltoets

Een eenvoudige statische webapp waarmee leerlingen online kunnen oefenen op:

- volgorde van kruisen en mollen;
- notennamen en wijzigingstekens;
- grote toonaarden herkennen;
- grote en kleine drieklanken herkennen.

## Lokaal openen

Open `index.html` direct in een browser, of start een lokale webserver:

```bash
python3 -m http.server 8000
```

Ga daarna naar <http://localhost:8000>.
